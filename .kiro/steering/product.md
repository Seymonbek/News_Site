# Product Overview

This is a multilingual news website built for an Uzbek-speaking audience. It publishes and organizes news articles across several categories and supports three languages: Uzbek (default), Russian, and English.

## Core Features

- News articles with categories, slugs, images, and publish status (Draft / Published)
- Category pages: Uzbekistan, Jahon (World), Sport, Fan_texnika (Science & Tech), Iqtisodiyot (Economy), Jamiyat (Society)
- Article detail pages with hit/view counting and authenticated user comments
- Full-text search across title and body
- Contact form for visitor inquiries
- User accounts: registration, login/logout, profile editing, password change/reset
- Superuser-only CRUD interface for creating, editing, and deleting news articles
- Django admin panel for content management

## Target Audience

Uzbek-speaking readers; content is primarily in Uzbek with Russian and English translations available via language switcher.
